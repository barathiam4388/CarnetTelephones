# Projet Final - Carnet de téléphones (Java)

Petit projet Java qui gère un carnet de téléphones avec validation et tests.

## Fichiers
- `CarnetTelephones.java` : logique principale du carnet
- `Telephone.java` : modèle / objet téléphone
- `TelephoneInvalideException.java` : exception personnalisée
- `TestsCarnetTelephones.java` : tests

## Exécuter
1. Ouvrir le dossier dans un IDE (IntelliJ / Eclipse / VS Code)
2. Lancer `TestsCarnetTelephones.java`
